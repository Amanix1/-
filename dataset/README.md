Dataset Information

This project uses the Basic Arabic Vocal Emotions Dataset (BAVED), which is publicly available on Kaggle and designed for Arabic speech and vocal pattern analysis research.

Dataset Link:
https://www.kaggle.com/datasets/a13x10/basic-arabic-vocal-emotions-dataset

The BAVED dataset contains Arabic speech recordings collected from multiple speakers with different vocal styles and expression levels. The recordings were designed to represent variations in speech characteristics such as pitch, tone, vocal energy, rhythm, and emotional vocal expression.

The dataset includes approximately 1935 Arabic audio recordings from different speakers. The speech samples are categorized into several speech intensity levels and vocal expression patterns, making the dataset suitable for speech signal analysis, machine learning, and deep learning applications. Some recordings contain emotional vocal expressions only, while others represent different speech intensity levels.

Dataset Categories:
	•	Low intensity speech
	•	Neutral intensity speech
	•	High intensity speech
